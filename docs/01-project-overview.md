# AYUR GATE -- Project Overview

## What is AYUR GATE?
AYUR GATE is a cloud-based, multi-tenant clinic management SaaS platform built specifically for Ayurveda and wellness clinics. It provides end-to-end management of patients, appointments, prescriptions, inventory, billing, and staff -- all from a single web application.

## Brand Distinction
- **AYUR GATE** -- The software platform (ayurgate.com)
- **Ayur Centre Pte. Ltd.** -- The Singapore-based clinic/company that developed it

## Target Users
- Ayurveda clinics in Singapore, India, Malaysia, Sri Lanka, and UAE
- Wellness and holistic therapy centres
- Multi-branch clinic chains
- Solo practitioners to mid-size clinics (1-50 staff)

## Current Status
- **Live at**: https://www.ayurgate.com
- **Stage**: Production (active users)
- **Business Model**: Freemium SaaS
  - 7-day free trial (no credit card required)
  - Starter plan: basic features
  - Professional plan: full features
  - Enterprise plan: custom pricing
- **Deployment**: Railway (auto-deploy from GitHub main branch)

## Key Differentiators
1. Built specifically for Ayurveda -- categories, treatments, and workflows tailored to traditional medicine
2. Multi-tenant from day one -- each clinic gets isolated data with a single deployment
3. PWA support -- installable on mobile devices, works offline-capable
4. Multi-branch support -- stock transfers, branch-level reporting
5. Doctor portal -- dedicated interface for doctors/therapists separate from admin
6. WhatsApp integration -- message templates for patient communication
7. Digital prescriptions -- PDF generation with medicine categories
